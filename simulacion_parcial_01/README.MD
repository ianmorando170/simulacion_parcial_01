### EJERCICIOS

**Ejercicio 1 — Validar Contraseña**
Escribí una función que reciba un string y retorne `True` si la contraseña cumple con todas las condiciones requeridas (longitud mínima, mayúscula, minúscula, dígito, sin espacios), o `False` si no las cumple.

**Ejercicio 2 — Combinar Dos Listas**
Escribí una función que una dos listas en una sola, recorriéndolas con un bucle (sin usar `+` ni `.extend()`).

**Ejercicio 3 — Eliminar Duplicados**
Escribí una función que elimine los elementos repetidos de una lista, conservando el orden de aparición (sin usar `set()`).

**Ejercicio 4 — Contar Apariciones**
Escribí funciones para contar cuántas veces aparece un elemento en una lista, y para mostrar el conteo de todos los elementos únicos (sin usar `.count()`).

**Ejercicio 5 — Número Primo**
Escribí una función que determine si un número es primo usando un bucle `while`.

**Ejercicio 6 — Conversor de Unidades con Módulo**
Desarrollá un conversor de kilómetros a millas y kilogramos a libras, con menú interactivo. Las funciones de conversión deben estar en un módulo separado `conversiones.py` con docstrings.

### Recomendaciones
(Ver también archivo Importante.md)

- Leer muy bien el enunciado de cada ejercicio.
- Siempre que puedan, hacer un pseudocódigo o un diagrama de flujo antes de empezar.
- Pensar el algoritmo antes de escribir código.
- Intentar hacer funciones pequeñas que resuelvan una sola cosa.
- Resolver primero que funcione, y luego refactorizar para que quede prolijo.
- En esa refactorización, evaluar si hacen falta validaciones del ingreso del usuario.
